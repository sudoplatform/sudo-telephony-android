provisionPhoneNumber

fun provisionPhoneNumber(countryCode: String, phoneNumber: String, sudoId: String, callback: (Result<PhoneNumber>) -> Unit): Unit

Provision a phone number to the signed in account. Searching for available phone numbers should be done before this call, ensuring that the passed in phone number is available.

Parameters

countryCode - Supported country of the number to provision. Code must be formatted in the ISO 3166-1 alpha-2 format. For example: US, BR, ND, etc.

phoneNumber - The E164 formatted phone number to provision.. For example: "+14155552671".

sudoId - The ID of the Sudo to provision the phone number for.

callback - Completion callback that provides the provisioned phone number.
