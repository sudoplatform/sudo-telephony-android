searchAvailablePhoneNumbers

fun searchAvailablePhoneNumbers(countryCode: String, limit: Int?, callback: (Result<PhoneNumberSearchResult>) -> Unit): Unit

Get available phone numbers for the specified country code.

Parameters

countryCode - Supported country code to search on. Code must be formatted in ISO 3166-1 alpha-2 format. For example: US, BR, ND, etc.

limit - Optional limit on the number of available phone numbers returned in the search result. If the limit is not specified, the default value is 10 and the search result will provide no more than 10 phone numbers.

callback - Completion callback providing a search result containing available phone numbers with the given country code.

fun searchAvailablePhoneNumbers(countryCode: String, prefix: String, limit: Int?, callback: (Result<PhoneNumberSearchResult>) -> Unit): Unit

Get available phone numbers for the specified country code and prefix.

Parameters

countryCode - Supported country code to search on. Code must be formatted in the ISO 3166-1 alpha-2 format. For example: US, BR, ND, etc.

prefix - Optional prefix (area code) used to limit the search results. If nil, only the country code will used for the search. If the country code and area code do not match, the resulting list of available phone numbers might be empty.

limit - Optional limit on the number of available phone numbers returned in the search result. If the limit is not specified, the default value is 10 and the search result will provide no more than 10 phone numbers.

callback - Completion callback providing a search result containing available phone numbers with the given country code and prefix.

Get available phone numbers for the specified coordinates.

Parameters

countryCode - Supported country code to search on. Code must be formatted in the ISO 3166-1 alpha-2 format. For example: US, BR, ND, etc.

callback - Completion callback providing a search result containing available phone numbers with the given country code and coordinates.
