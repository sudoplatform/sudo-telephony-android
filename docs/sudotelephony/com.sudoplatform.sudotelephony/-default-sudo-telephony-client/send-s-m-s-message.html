fun sendSMSMessage(localNumber: PhoneNumber, remoteNumber: String, body: String, callback: (Result<PhoneMessage>) -> Unit): Unit

Sends an SMS message from a provisioned phone number to another number

Parameters

localNumber - The E164 formatted phone number to send the message from. For example: "+14155552671".

remoteNumber - The E164 formatted phone number of the recipient. For example: "+14155552671".

body - The body of the SMS message

callback - Completion callback providing the sent message or an error if there was a failure.
