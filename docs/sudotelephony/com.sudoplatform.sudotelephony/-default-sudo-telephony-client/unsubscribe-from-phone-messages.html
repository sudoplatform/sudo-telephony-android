unsubscribeFromPhoneMessages

fun unsubscribeFromPhoneMessages(id: String?): Unit

Unsubscribes the specified subscriber so that it no longer receives notifications about new 'PhoneMessage' objects. If no id is provided, all subscribers will be unsubscribed.

Parameters

id - unique ID for the subscriber.
