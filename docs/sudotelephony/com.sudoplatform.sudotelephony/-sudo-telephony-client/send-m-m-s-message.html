sendMMSMessage

abstract fun sendMMSMessage(localNumber: PhoneNumber, remoteNumber: String, body: String, localUrl: URL, callback: (Result<PhoneMessage>) -> Unit): Unit

Sends an MMS message from a provisioned phone number to another number

Parameters

localNumber - The provisioned phone number to send the message from

remoteNumber - The E164 formatted phone number of the recipient. For example: "+14155552671".

body - The body of the MMS message

localUrl - The local path of the media to be uploaded.

callback - Completion callback providing the sent message or an error if there was a failure.
