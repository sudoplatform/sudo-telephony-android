connectionStatusChanged

abstract fun connectionStatusChanged(state: ConnectionState): Unit

Notifies the subscriber that the subscription connection state has changed. The subscriber won't be notified until the connection status changes to ConnectionState.CONNECTED. The subscriber will stop receiving notifications when the connection state changes to ConnectionState.DISCONNECTED.
